# 博学者·通识学习 - 逻辑学课程增强版

## 项目状态
已完成逻辑学课程的增强工作：

### 主要更新
1. **小节测验增强**：为逻辑学全部50个小节（L-A01到L-E10）每节增加了5道题目
2. **总题量统计**：逻辑学课程现在拥有250道小节测验题 + 50道章节测验题 = 300道总题目
3. **结构优化**：将所有小节测验整合到统一的lesson-quizzes.ts文件中

### 详细说明
- **小节测验**：50个小节 × 5题/节 = 250道小节测验题
- **章节测验**：5个章节 × 10题/章 = 50道章节测验题（原有基础上翻倍）
- **总计**：300道逻辑学测验题

## 文件结构变更
- `src/data/logic/quizzes/lesson-quizzes.ts` - 整合了全部50个小节的测验
- `src/data/logic/index.ts` - 更新以包含小节测验
- 移除了独立的小节测验文件以优化结构

## 重新构建项目
如果您需要运行此应用，请按以下步骤操作：

1. **安装依赖**：
   ```bash
   npm install
   ```

2. **构建项目**：
   ```bash
   npm run build
   ```

3. **运行开发服务器**：
   ```bash
   npm run dev
   ```

4. **或直接运行桌面应用**：
   ```bash
   python launch.py
   ```
   
   或者：
   ```bash
   npm run desktop
   ```
